Custom Webpack Configurations In Angular
There are a few steps to custom Webpack. Suppose that we want to use the webpack-bundle-analyzer package to analyze the bundles.
  1. Install required packages.
  2. Update angular.json.
  3. Add the webpack.config.js.
  4. Build the bundle.
  5. Execute the Webpack build.
  6. It will open a browser on http://127.0.0.1.
