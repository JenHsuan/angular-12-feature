Introduction
There are a few improvements of stylishment in Angular 12.
  1. Supports Inline Sass in components
  2. Support new Sass API for Angular CDK and Angular Material code
  3. Support TailwindCSS JIT node (Just-In-Time)
Inline Sass in components
It's helpful for using single file component. Previously, Sass was only available in external resources due to the Angular compiler.

Use TailwindCSS in Angular
TailwindCSS is an utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.
There are some reasons to use TailwindCSS instead of the traditional CSS.
There are steps to set up the TailwindCSS in Angular.
  1. Install required packages.
  2. Generate tailwind.config.js.
  3. Update the style.scss.
  4. Update the tailwind.config.js.
