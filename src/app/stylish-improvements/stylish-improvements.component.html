There are a few advantages for using TailwindCSS instead of the traditional CSS.
  1. TailwindCSS provides many pre-defined classes. It's clear to understand what's the effect of the CSS style class.
  2. The styles between elements will be consistent because there is a global configuration file, which is better than the inline CSS.
  3. We are allowed to write psuedo class and the media class, which is better than the inline CSS.
In Angular 12, we can use the JIT (Just-In-Time) mode to generate CSS files when writing HTML files.
  1. It reduces the loading time.
  2. We are allowed to use psuedo elements without override variants.
  3. We are allowed to create a new class with any values in the HTML. (e.g., top-[10px])
