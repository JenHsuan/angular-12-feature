Introduction
Angular language service is an IDE extension that allows code editors to get helpful tips, code completion navigation. In v12, they unleash the power of Ivy to help boost developer velocity when usiing language service plugin.
  • Install the language service plugin: Extensions -> Search "language service"
New Feature
Be able to check the symbol references from the template directly
Old Features
  1. Syntax highlighting
    Highlight the syntax in the template.
  2. Go to/Peek definition
  3. Type detail on hover
