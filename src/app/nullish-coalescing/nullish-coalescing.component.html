Introduction
Baically, we usually use the ternary operator to declare a condition. Then we assign the value for the corresponding result of the condition.
  • example:
  • result:
    {{age !== null && age !== undefined ? age : calculateAge() }}
In v12, we are allowed to use the Nullish Coalescing on the template.
  • example:
  • result:
    {{ age ?? calculateAge() }}
